# Supabase Setup

## 1. Run schema & seed in Supabase

1. Open [Supabase Dashboard](https://supabase.com/dashboard) → your project
2. **SQL Editor** → **New query**
3. Copy-paste `schema.sql` → **Run**
4. New query again → copy-paste `seed.sql` → **Run**
